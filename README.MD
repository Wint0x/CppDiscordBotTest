Test C++ Discord bot application. Made using the D++ Discord API library.  
# Requirements  
> D++<br>
> C++ 20<br>    
## To compile  
`g++ -std=c++20 main.cpp -o bot.out -ldpp`  

**Target Platform**: Linux Mint  
D++ built from source  
```sh
git clone https://github.com/brainboxdotcc/DPP  
cd DPP  
mkdir build  
cmake -B ./build  
cd build  
sudo make install  
```
